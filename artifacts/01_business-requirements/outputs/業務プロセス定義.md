### 業務プロセス：BIZ-001 申請種別判断・案内

#### 基本情報
- 業務ID：BIZ-001
- 業務名：申請種別判断・案内
- 業務目的：社員が入力した申請内容から必要な申請種別（交通費精算申請・経費精算申請）を判断して提示する。判断できない場合はユーザーに選択肢を提示して確認する
- 対象ユーザ：一般社員
- 開始条件（トリガー）：社員が申請内容を入力する
- 終了条件：申請種別が確定し、対応する申請書作成フローへ誘導される

#### 業務フロー（To-Be）

```mermaid
flowchart TD
    A[社員：申請内容を入力] --> B[AI：申請内容を解析]
    B --> C{AI：申請種別を判断}
    C -->|交通費精算申請| D[AI：交通費精算申請を案内]
    C -->|経費精算申請| E[AI：経費精算申請を案内]
    C -->|判断不能| F[AI：選択肢を提示]
    C -->|対象外| G[AI：担当者へのエスカレーションを案内]
    F --> H[社員：申請種別を選択]
    H --> I[AI：選択された申請種別を案内]
    D --> J[申請書作成フローへ]
    E --> J
    I --> J
    G --> K[業務終了]
```

### 業務プロセス：BIZ-002〜003 申請書作成（共通フロー）

#### 業務フロー（To-Be）

```mermaid
flowchart TD
    A[申請書作成開始] --> B[AI：必要情報を確認]

    C -->|不足あり| D[AI：不足情報を対話で確認]
    D --> E[社員：不足情報を回答]
    E --> C
    C -->|情報揃い| F{AI：申請期限チェック}

    I --> K[社員：申請書を確認]
```

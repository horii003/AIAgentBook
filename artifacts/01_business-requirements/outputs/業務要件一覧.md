## ターゲットユーザー
社内の一般社員（申請ルールや申請方法に詳しくない場合がある）

## 要求事項
- RQ-01: 申請内容を入力すると、必要な申請種別・申請書・申請先を提示する
- RQ-02: 社内申請ルールを参照して申請種別を自動判断する
- RQ-03: 不足情報を対話で確認しながら申請書を自動作成する
- RQ-04: 申請ミスや差し戻しを減らす
- RQ-05: 申請ルールや申請方法に詳しくない利用者でも正しく申請できる

## 対象申請種別

本システムは以下の2種類の申請のみを対象とする。

| 申請種別 | 申請書名 | 概要 |
| -------- | -------- | ---- |
| 交通費精算申請 | 交通費精算申請書 | 業務上発生した交通費（電車・バス・タクシー・飛行機）の精算 |
| 経費精算申請 | 経費精算申請書 | 業務上発生した経費（事務用品費・宿泊費・資格精算費・その他経費）の精算 |

## 業務種別ごとの業務ルール

### 交通費精算申請の業務ルール

| ID | 業務ルール | 詳細 | 優先度 |
| -- | ---------- | ---- | ------ |
| TRV-01 | 交通費精算申請の適用条件 | 業務上発生した交通費（電車・バス・タクシー・飛行機）の精算に使用する | High |
| TRV-02 | 対象交通手段 | 電車、バス、タクシー、飛行機の4種類 | High |
| TRV-03 | 情報収集方法 | 移動情報（移動日、出発地、目的地、交通手段、費用、業務目的）を一区間ずつ収集する | High |
| TRV-04 | 交通費自動計算 | 運賃データから交通費を自動計算する（電車は経路テーブル検索、バス/タクシー/飛行機は固定運賃） | High |
| TRV-05 | 申請期限 | 申請は経費発生日から3ヶ月以内であること | High |
| TRV-06 | 上長承認閾値 | 交通費が10,000円を超える場合は事前に上長の承認が必要 | High |
| TRV-07 | 駅名正規化 | 駅名の正規化を行う（「渋谷駅」→「渋谷」） | Medium |
| TRV-08 | 業務目的必須 | すべての申請に業務目的の記載が必要 | High |
| TRV-09 | 申請先 | 所属部門の経理担当者 | High |

### 経費精算申請の業務ルール

| ID | 業務ルール | 詳細 | 優先度 |
| -- | ---------- | ---- | ------ |
| EXP-01 | 経費精算申請の適用条件 | 業務上発生した経費（交通費を除く）の精算に使用する | High |
| EXP-02 | 対象経費区分 | 事務用品費（書籍、文房具、オフィス用品など）、宿泊費（ホテル、宿泊施設など）、資格精算費（資格試験、受験料、認定費用など）、その他経費（上記以外の業務関連経費）の4区分 | High |
| EXP-03 | 領収書画像からの自動抽出 | 領収書画像から経費情報（購入日、店舗名、品目、金額）を品目ごとに自動抽出する | High |
| EXP-04 | 経費区分自動判断 | 品目から経費区分を自動判断する（事務用品費、宿泊費、資格精算費、その他経費の4区分） | High |
| EXP-05 | 申請期限 | 申請は経費発生日から3ヶ月以内であること | High |
| EXP-06 | 上長承認閾値 | 経費が5,000円を超える場合は事前に上長の承認が必要 | High |
| EXP-07 | 業務目的必須 | すべての申請に業務目的の記載が必要 | High |
| EXP-08 | 申請先 | 所属部門の経理担当者 | High |

## 申請種別の判断不能ケース

申請内容から交通費精算申請・経費精算申請のいずれかを判断できない場合：
- ユーザーに「交通費精算申請」「経費精算申請」の選択肢を提示する
- ユーザーの選択に従って対応する申請フローへ誘導する

## 申請例

| 申請内容 | 判断結果 |
| -------- | -------- |
| 「タクシー代を精算したい」 | 交通費精算申請 |
| 「電車で移動した交通費を精算したい」 | 交通費精算申請 |
| 「文房具を購入した領収書を精算したい」 | 経費精算申請 |
| 「ホテルの宿泊費を精算したい」 | 経費精算申請 |

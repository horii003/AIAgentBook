# ログ出力要件定義

> **参照元:**
> - 業務プロセス定義.md（ログ出力が必要な処理の特定）
> - エージェント一覧.md（ログ出力主体の特定）
> - 会話フロー一覧.md（ログ出力タイミングの特定）

## 1. ログ出力方針

| 項目 | 内容 |
|---|---|
| ログフレームワーク | Python標準logging |
| ログフォーマット | `%(asctime)s [%(levelname)s] %(name)s - %(message)s` |
| ログファイルエンコーディング | UTF-8 |
| ログローテーション | RotatingFileHandler: 10MB × 5世代 |

## 2. ログ出力先定義

| No | ログ種別 | 出力先 | ログレベル | 用途 |
|---:|---|---|---|---|
| 1 | アプリケーションログ | logs/app.log | INFO以上 | 正常処理の記録、デバッグ情報 |
| 2 | エラーログ | logs/error.log | ERROR以上 | エラー発生の記録 |

## 3. ログレベル定義

| レベル | 用途 | 出力例 |
|---|---|---|
| DEBUG | 開発時のデバッグ情報 | 変数値、処理経過 |
| INFO | 正常処理の記録 | 処理開始、処理完了、データ読み込み成功 |
| WARNING | 注意が必要な状態 | ループ制限到達 |
| ERROR | エラー発生 | バリデーションエラー、ファイル読み込み失敗、予期しないエラー |

## 4. ログ出力対象イベント

| No | イベント | ログレベル | 記録内容 | 対象エージェント |
|---:|---|---|---|---|
| 1 | エージェント処理開始 | INFO | エージェントID、セッションID | AG-001〜AG-003 |
| 2 | エージェント処理完了 | INFO | エージェントID、処理結果概要 | AG-001〜AG-003 |
| 3 | 申請種別判断完了 | INFO | 判断結果、適用ルール | AG-001 |
| 4 | ツール実行開始 | INFO | ツールID、入力パラメータ概要 | AG-002, AG-003 |
| 5 | ツール実行完了 | INFO | ツールID、実行結果概要 | AG-002, AG-003 |
| 6 | 申請書生成完了 | INFO | ファイルパス、申請種別 | AG-002, AG-003 |
| 7 | バリデーションエラー | ERROR | エラー内容、対象データ | AG-001〜AG-003 |
| 8 | ファイル読み込み失敗 | ERROR | ファイルパス、エラー原因 | AG-002, AG-003 |
| 9 | ループ制限到達 | WARNING | エージェントID、ループ回数 | AG-001〜AG-003 |
| 10 | 予期しないエラー | ERROR | エラー内容、スタックトレース | AG-001〜AG-003 |

## 5. ログ出力時の制約

| 項目 | 内容 |
|---|---|
| 個人情報のマスキング | 申請者名等の個人情報はログに出力しない、またはマスキングする |
| スタックトレース | ERROR時は`exc_info=True`でスタックトレースを付与 |
| Strandsイベントループログ | WARNINGレベルに設定（過剰なデバッグ出力を抑制） |
| ログ出力の責務 | ErrorHandlerはメッセージ生成のみ。ログ出力は呼び出し元が行う |

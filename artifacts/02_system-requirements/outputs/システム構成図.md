```mermaid
flowchart TD
    User["👤 一般社員\n（CLIユーザー）"]

    subgraph App["申請支援AIエージェントシステム"]
        Orch["AG-001\n申請受付窓口エージェント\n（オーケストレーター）"]
        TrvAgent["AG-002\n交通費精算申請エージェント\n（専門エージェント）"]
        ExpAgent["AG-003\n経費精算申請エージェント\n（専門エージェント）"]
    end

    subgraph Tools["ツール層"]
        CalcTool["TOOL-001\n交通費計算ツール"]
        GenTool["TOOL-002\n申請書生成ツール"]
    end

    subgraph Data["データ層（ローカルファイル）"]
        TrainData["data/train_fares.json\n電車経路テーブル"]
        FixedData["data/fixed_fares.json\n固定運賃データ"]
        TrvTemplate["template/交通費精算申請書.xlsx\n申請書テンプレート"]
        ExpTemplate["template/経費精算申請書.xlsx\n申請書テンプレート"]
        Output["output/\n生成済み申請書"]
        Sessions["storage/sessions/\nセッションデータ"]
        Logs["logs/\nアプリログ・エラーログ"]
    end

    subgraph AWS["Amazon Web Services"]
        Bedrock["Amazon Bedrock\nLLMバックエンド\n（Claude Sonnet）"]
    end

    User -->|"申請内容入力"| Orch
    Orch -->|"申請種別：交通費精算申請\n（Agent as Tool）"| TrvAgent
    Orch -->|"申請種別：経費精算申請\n（Agent as Tool）"| ExpAgent
    TrvAgent -->|"運賃計算"| CalcTool
    TrvAgent -->|"申請書生成"| GenTool
    ExpAgent -->|"申請書生成"| GenTool
    CalcTool -.->|"参照"| TrainData
    CalcTool -.->|"参照"| FixedData
    GenTool -.->|"参照"| TrvTemplate
    GenTool -.->|"参照"| ExpTemplate
    GenTool -->|"生成"| Output
    Orch <-->|"LLM推論"| Bedrock
    TrvAgent <-->|"LLM推論"| Bedrock
    ExpAgent <-->|"LLM推論"| Bedrock
    App -.->|"セッション保存"| Sessions
    App -.->|"ログ出力"| Logs
    Output -->|"申請書確認・提出"| User
```

**凡例：**
- 実線矢印（→）：処理の委譲・呼び出し・データの生成
- 破線矢印（-.->）：データの参照・取得・保存

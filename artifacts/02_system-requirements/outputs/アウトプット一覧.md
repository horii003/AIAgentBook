## 2. 成果物詳細定義（成果物ID単位）

### 2.1 成果物ID：OUT-005
### 成果物名：交通費精算申請書

---

#### 2.1.1 目的・位置づけ
- 業務上の目的：交通費精算申請のために経理部門へ提出するExcel形式の申請書
- 業務ゴールとの関係：BIZ-002の最終成果物。申請書提出により交通費精算申請が完了する
- 最終成果物である理由：申請者が経理部門へ提出する最終的な申請書類

---

#### 2.1.2 成果物概要
- 利用シーン：交通費精算申請書作成フロー（CF-002）の完了時
- 利用者：申請者（一般社員）
- 想定される業務アクション：経理部門への提出

---

#### 2.1.3 生成条件・トリガー

| 項目 | 内容 |
|---|---|
| 生成トリガー | ユーザーが申請書ドラフトを確認しOKを選択する（JD-10） |
| 前提条件 | 移動情報（D-004〜D-008）・交通費計算結果（D-015）が収集済みであること |
| 生成頻度 | 都度（申請ごと） |

---

#### 2.1.4 生成主体・責任分界

| 項目 | 内容 |
|---|---|
| 主生成エージェント | AG-002（交通費精算申請エージェント） |
| 協調/委譲エージェント | — |
| 最終責任者 | 申請者（一般社員） |
| エージェント自律度 | Lv4（実行＋承認依頼） |

---

#### 2.1.5 入力定義

| No | 入力名 | 種別 | 出所 | 必須 | 備考 |
|---:|---|---|---|---|---|
| 1 | 申請者名 | セッション情報 | invocation_state | ○ | |
| 2 | 移動日 | ユーザー入力 | D-004 | ○ | |
| 3 | 出発地 | ユーザー入力 | D-005 | ○ | 正規化済み |
| 4 | 目的地 | ユーザー入力 | D-006 | ○ | 正規化済み |
| 5 | 交通手段 | ユーザー入力 | D-007 | ○ | |
| 6 | 交通費 | 計算結果 | D-015 | ○ | |
| 7 | 業務目的 | ユーザー入力 | D-008 | ○ | |
| 8 | 申請書テンプレート | マスタデータ | D-022 | ○ | |

---

#### 2.1.6 出力仕様

- 種別：Excelファイル（.xlsx）
- 文字コード：UTF-8
- ファイル命名規則：`交通費精算申請書_{YYYYMMDD_HHMMSS}.xlsx`
- 保存先：`output/`ディレクトリ

#### 2.1.7 申請書テンプレートのセル位置定義

| 項目 | セル位置 | 備考 |
|---|---|---|
| 申請者名 | B3 | |
| 申請日 | B4 | エージェント側で設定（申請者への確認不要） |
| 移動区間（No/日付/出発地/目的地/交通手段/費用/業務目的/承認状況） | A{7+i}〜H{7+i} | i=0〜n-1（nは件数） |
| 合計金額 | H{7+n+2} | |

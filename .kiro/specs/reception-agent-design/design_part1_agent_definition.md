# 申請受付窓口エージェント 設計書

# 1. エージェント定義

## 1-1. 名称

- **正式名称**: 申請受付窓口エージェント
- **英語名**: Reception Agent
- **クラス名**: `ReceptionAgent`
- **エージェントID**: `reception_agent`

## 1-2. 使用モデル

- **モデル名**: Claude Sonnet 4.5
- **モデルID**: `jp.anthropic.claude-sonnet-4-5-20250929-v1:0`
- **プロバイダー**: Amazon Bedrock (Anthropic)

## 1-3. 役割

申請受付窓口エージェントは、社内申請プロセスのオーケストレーターとして機能します。ユーザーからの申請内容を受け付け、その内容を分析し、適切な専門エージェント（交通費精算申請エージェント、経費精算申請エージェント）に自動的に振り分けます。

申請ルールや申請方法が分からない社員でも、自然な日本語で申請内容を入力するだけで、正しく申請できるようにすることを目的としています。

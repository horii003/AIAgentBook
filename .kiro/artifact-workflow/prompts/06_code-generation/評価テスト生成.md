# 役割
あなたは Strands Agents SDK で構築したエージェントの品質を自動評価するスクリプトの実装を担当する開発者です。

# 指示
設計書と実装コードを読み込み、テンプレート（18_eval_test.md）のプレースホルダーを埋めて評価コードを生成してください。

- 規約（`.kiro/steering/00_rule_project_conventions.md`）に従うこと

# 実行手順

## Step 1: helpers.py の生成

テンプレート Part 1 を基に `evals/helpers.py` を生成する。
実装コード（src/）のエージェント構成・フック・状態管理の実装を読み、
設計書の方針に従って TODO を実装で埋める。

## Step 2: eval_*.py の生成

テンプレート Part 2（マルチターン）または Part 3 の差分を適用（シングルターン）して
`evals/eval_{evaluation_name}.py` を生成する。プレースホルダーは設計書の対応する記載から読み取る。

## Step 3: テストケースの記入

設計書のケース一覧テーブルの全行を `EVAL_CASES` に転記する。
